# 🚀 快速开始指南

## 📥 安装步骤

### 1. 克隆仓库
```bash
git clone https://github.com/ahouhou/game.git
cd game
```

### 2. 安装依赖
```bash
# 使用 pip 安装
pip install -r requirements.txt

# 或使用 pip3
pip3 install pygame
```

### 3. 运行游戏

#### GUI 版本（推荐）
```bash
python3 survival_game_gui.py
```

#### 文字版本
```bash
python3 survival_game.py
```

## 🎯 游戏目标

**存活30天即可通关！**

### 生存要点
1. **管理资源**: 平衡食物、材料、工具的使用
2. **建造建筑**: 建筑可以提供保护和功能
3. **应对灾害**: 灾害会造成伤害，需要提前准备
4. **维持生命**: 保持生命值、饱食度、体力在安全范围

## 💡 游戏技巧

### 早期策略（第1-10天）
1. 优先建造小木屋（成本低，效果好）
2. 多进行捕捞和探索积累资源
3. 不要浪费食物，合理分配

### 中期策略（第11-20天）
1. 建造瞭望塔降低灾害概率
2. 建造防御墙减免灾害伤害
3. 开始建造冶炼屋和工坊

### 后期策略（第21-30天）
1. 建造仓库增加存储
2. 准备充足的食物
3. 保持建筑完整度

## 🐛 常见问题

### Q: 游戏无法启动？
A: 确保已安装 Pygame，运行 `pip install pygame`

### Q: 如何保存游戏？
A: 文字版本支持自动保存，GUI版本预留了保存功能

### Q: 如何重新开始？
A: 关闭游戏重新运行即可

### Q: 灾害太频繁？
A: 建造瞭望塔可以降低灾害概率
