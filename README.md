# 🏝️ 荒岛求生 - Survival Island Game

一款文字冒险生存游戏，玩家需要在荒岛上生存、建造、制造装备并抵御各种威胁。

## 🎮 游戏特色

- **🎣 撒网捕捞**: 捕捞海中物资，获取食物和材料
- **🏗️ 建造系统**: 建造小木屋、石屋、瞭望塔、防御墙等建筑
- **🔨 制造系统**: 制造工具、武器、防具
- **⚔️ 战斗系统**: 对抗海蟹、鲨鱼、巨型章鱼、海龙王等敌人
- **⚠️ 灾害系统**: 面对暴风雨、海啸、瘟疫、龙卷风等自然灾害
- **🏆 成就系统**: 生存大师、渔夫、战士等多种成就

## 🎯 游戏目标

存活 **30天** 即可通关！

## 📖 游戏玩法

### 基础操作
| 行动 | 说明 |
|------|------|
| 撒网捕捞 | 获取食物和材料 |
| 探索岛屿 | 发现资源，可能遭遇敌人 |
| 进食 | 恢复饱食度 |
| 治疗 | 使用草药恢复生命 |
| 休息 | 恢复体力 |

### 制造系统
- **工具**: 石斧、鱼竿、撒网、高级撒网
- **武器**: 木棍、石矛、鱼叉、三叉戟(传说)
- **防具**: 树叶衣、兽皮甲、贝壳甲、海神甲(传说)

### 建筑系统
- **小木屋**: 提升休息效率
- **石屋**: 更好的保护效果
- **瞭望塔**: 降低灾害概率
- **防御墙**: 减免灾害伤害
- **仓库/工坊**: 解锁更多功能

### 敌人
| 敌人 | HP | 攻击 | 掉落 |
|------|-----|------|------|
| 海蟹 | 20 | 5 | 石头、绳索 |
| 鲨鱼 | 50 | 15 | 鱼、金属 |
| 巨型章鱼 | 80 | 20 | 绳索、金属 |
| 海蛇 | 40 | 25 | 草药、金属 |
| 海龙王 | 150 | 35 | 金属、神秘果实 |

## 🚀 运行游戏

```bash
python3 survival_game.py
```

## 📦 依赖

- Python 3.6+
- 无需额外依赖，使用标准库
