# 🏝️ 荒岛求生 - Survival Island Game

一款完整的图形界面生存游戏，玩家需要在荒岛上生存、建造、制造装备并抵御各种威胁。

## 🎮 游戏特色

### 核心系统
- **🎣 撒网捕捞**: 点击按钮捕捞海中物资，获取食物和材料
- **🏗️ 建造系统**: 建造7种不同的建筑（小木屋、石屋、瞭望塔、防御墙、冶炼屋、仓库、工坊）
- **⚠️ 灾害系统**: 面对暴风雨、海啸、龙卷风、瘟疫等自然灾害，每种灾害都有独特的视觉特效
- **🎨 粒子特效**: 
  - 捕捞时的水花特效
  - 建造时的爆发特效
  - 灾害时的环境特效（雨、浪、龙卷风、瘟疫）
- **📊 实时UI**: 生命、饱食度、体力状态条，背包、建筑、消息实时显示

### 游戏机制
- **日夜循环**: 每天有3个行动点，用完后进入下一天
- **资源管理**: 管理食物、材料、工具等资源
- **建筑效果**: 不同建筑提供不同的保护和功能
- **灾害伤害**: 灾害会造成生命值损失，建筑可以减免伤害

## 🎯 游戏目标

存活 **30天** 即可通关！


## 🎮 游戏流程

1. **初始化**: 开始时有基础物资（木材、石头、鱼）
2. **日常活动**: 每天有3个行动点，可以捕捞、探索、进食、建造
3. **资源管理**: 收集资源，制造工具和建筑
4. **灾害应对**: 随机灾害会造成伤害，建筑可以提供保护
5. **生存目标**: 存活30天即可通关

## 🏆 成就系统

- 🏆 生存大师：存活10天
- 🏆 渔夫：捕捞50次
- 🏆 战士：击败10个敌人
- 🏆 传说：获得三叉戟
- 🏆 铁壁：获得海神甲
