## 🎮 游戏特色

### 1. 完整的生存系统
- **资源管理**: 木材、石头、金属、布料、绳索、草药等10+种资源
- **食物系统**: 鱼、椰子、烤鱼等多种食物
- **工具装备**: 工具、武器、防具等装备系统

### 2. 建筑系统（7种建筑）
| 建筑 | 特色 | 成本 |
|------|------|------|
| 小木屋 | 基础庇护所 | 低 |
| 石屋 | 坚固住所 | 中 |
| 瞭望塔 | 预警设施 | 中 |
| 防御墙 | 防御设施 | 高 |
| **冶炼屋** | 金属冶炼 | 高 |
| 仓库 | 存储设施 | 中 |
| 工坊 | 制造设施 | 高 |

### 3. 灾害系统（4种灾害）
每种灾害都有独特的视觉特效：
- 🌧️ 暴风雨 - 灰色雨滴粒子
- 🌊 海啸 - 蓝色波浪粒子
- 🌪️ 龙卷风 - 旋转风暴粒子
- 💜 瘟疫 - 紫色雾气粒子

### 4. 高质量特效
- **粒子系统**: 通用粒子引擎，支持各种特效
- **灾害特效**: 每种灾害都有沉浸式的视觉表现
- **交互反馈**: 操作时有粒子特效反馈

## 🎨 设计亮点

### 1. 视觉设计
- **色彩方案**: 海洋蓝、沙滩黄、森林绿、石头灰的和谐搭配
- **UI布局**: 左背包、中岛屿、右建筑的三分布局
- **特效美学**: 每种灾害都有独特的粒子特效

### 2. 交互设计
- **鼠标操作**: 5个主要操作按钮
- **键盘快捷键**: 1-7快速选择建筑
- **反馈机制**: 视觉、文字、粒子多重反馈

### 3. 游戏平衡
- **难度曲线**: 前期简单，后期困难
- **资源平衡**: 捕捞、探索、建造的成本收益平衡
- **灾害平衡**: 不同灾害的伤害和概率平衡

## 🚀 使用指南

### 安装
```bash
git clone https://github.com/ahouhou/game.git
cd game
pip install -r requirements.txt
```

### 运行
```bash
# GUI版本（推荐）
python3 survival_game_gui.py

# 文字版本
python3 survival_game.py
```
