# 🏝️ 荒岛求生 - 游戏设计文档

## 📋 项目概述

**游戏名称**: 荒岛求生 (Survival Island)  
**游戏类型**: 单机生存模拟游戏  
**开发工具**: Python + Pygame  
**目标平台**: Windows / macOS / Linux  
**游戏时长**: 30分钟 - 1小时

---

## 🎮 核心玩法

### 游戏目标
在荒岛上存活30天，通过捕捞、探索、建造和制造来维持生存。

### 核心机制

#### 1. 日夜循环系统
- **每天3个行动点**，用完后进入下一天
- **自动扣除资源**：饥饿度每天下降15点，生命值随之下降
- **随机灾害**：30%概率触发灾害事件

#### 2. 资源管理
- **基础资源**: 木材、石头、金属、布料、绳索、草药
- **食物**: 鱼、椰子、烤鱼、海龟肉
- **工具**: 石斧、鱼竿、撒网、高级撒网
- **装备**: 武器（木棍、石矛、鱼叉、三叉戟）、防具（树叶衣、兽皮甲、贝壳甲、海神甲）

#### 3. 建筑系统
玩家可以建造7种建筑，每种建筑有不同的功能和成本：

| 建筑 | 成本 | 功能 | 视觉特征 |
|------|------|------|---------|
| 小木屋 | 木材x20, 绳索x5 | 基础庇护所 | 棕色木质结构 |
| 石屋 | 石头x30, 木材x10 | 坚固住所 | 灰色石质结构 |
| 瞭望塔 | 木材x15, 石头x10 | 预警设施 | 金色塔形结构 |
| 防御墙 | 石头x25, 木材x10, 金属x5 | 防御设施 | 深灰色墙体 |
| **冶炼屋** | 石头x20, 金属x5, 木材x15 | 金属冶炼 | 橙红色炉火 |
| 仓库 | 木材x20, 石头x10 | 存储设施 | 浅棕色仓库 |
| 工坊 | 木材x25, 石头x20, 金属x10 | 制造设施 | 银色工业风格 |

#### 4. 灾害系统
随机灾害会对玩家造成伤害，每种灾害都有独特的视觉特效：

| 灾害 | 伤害 | 概率 | 特效描述 |
|------|------|------|---------|
| 暴风雨 | 15点 | 15% | 灰色雨滴粒子从上方落下 |
| 海啸 | 25点 | 8% | 蓝色波浪粒子向外扩散 |
| 龙卷风 | 20点 | 5% | 灰色粒子旋转形成风暴 |
| 瘟疫 | 18点 | 12% | 紫色雾气粒子漂浮 |

## ✨ 特效系统

### 1. 粒子系统
- **捕捞特效**: 蓝色水花粒子向外扩散
- **探索特效**: 绿色粒子表示发现
- **建造特效**: 建筑颜色的爆发粒子
- **进食特效**: 橙色粒子表示补充能量

### 2. 灾害特效

#### 暴风雨特效
```
实现方式: 灰色粒子从上方持续落下
- 粒子数量: 30个初始 + 每帧2个新增
- 粒子速度: 水平3-6 px/s, 垂直3-6 px/s
- 持续时间: 2秒
- 视觉效果: 模拟下雨场景
```

#### 海啸特效
```
实现方式: 蓝色粒子向外扩散
- 粒子数量: 50个
- 粒子速度: 8 px/s (向外辐射)
- 持续时间: 1.5秒
- 视觉效果: 波浪冲击
```

#### 龙卷风特效
```
实现方式: 灰色粒子旋转形成风暴
- 粒子数量: 40个初始 + 每帧3个新增
- 粒子速度: 旋转运动 + 向上升起
- 持续时间: 1.5秒
- 视觉效果: 旋转风暴
```

#### 瘟疫特效
```
实现方式: 紫色雾气粒子漂浮
- 粒子数量: 30个
- 粒子速度: 缓慢漂浮 (-1 to 1 px/s)
- 持续时间: 1.5秒
- 视觉效果: 神秘雾气
```

## 🎮 交互设计

### 鼠标交互
- **按钮点击**: 5个主要操作按钮
- **悬停效果**: 按钮边框高亮
- **反馈**: 操作成功时显示消息和粒子特效

### 键盘快捷键
- **1-7**: 快速选择建筑类型
- **ESC**: 返回菜单（预留）
- **P**: 暂停游戏（预留）

### 反馈机制
- **视觉反馈**: 粒子特效、颜色变化
- **文字反馈**: 消息日志显示操作结果
- **音效反馈**: 预留音效系统

## 📊 游戏平衡性

### 难度曲线
- **第1-10天**: 简单，资源充足，灾害较少
- **第11-20天**: 中等，资源竞争，灾害频繁
- **第21-30天**: 困难，资源稀缺，灾害严重

### 资源平衡
- **捕捞收益**: 鱼x2-5, 木材x0-2, 石头x0-2
- **探索收益**: 木材x3-6 或 石头x2-5 或 椰子x1-3
- **消耗速率**: 每天饥饿度-15, 建造消耗大量资源

## 🔧 技术架构

### 核心类结构
```
SurvivalGame (主游戏类)
├── Player (玩家数据)
├── ParticleSystem (粒子系统)
├── DisasterEffect (灾害特效)
└── GameData (游戏数据)
```

### 主要方法
- `fish()` - 捕捞逻辑
- `explore()` - 探索逻辑
- `build()` - 建造逻辑
- `next_day()` - 日期推进
- `draw()` - 渲染所有内容
- `update()` - 更新游戏状态
